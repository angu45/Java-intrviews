01 Basic Syntax & Types

Main Method Template

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

Primitive Data Types

  • int 4 bytes (-2B to 2B)
  • double 8 bytes (Decimals)
  • boolean 1 bit (true/false)
  • char 2 bytes (Unicode)

02 OOP Pillars

Inheritance

Use extends for classes and implements for interfaces.

class Dog extends Animal {
  @Override
  void sound() { System.out.println("Bark"); }
}

Encapsulation

Private fields + Public getters/setters.

private String name;
public String getName() { return name; }
public void setName(String n) { name = n; }

03 Collections Framework

Structure Implementation Key Characteristics
List ArrayList Fast random access, duplicate allowed.
Set HashSet Unique elements, no order.
Map HashMap Key-Value pairs, unique keys.

04 Java 8+ Streams 

List<String> names = Arrays.asList("Java", "Python", "C++");

List<String> filtered = names.stream()
    .filter(s -> s.startsWith("J"))
    .map(String::toUpperCase)
    .sorted()
    .collect(Collectors.toList());

// Output: ["JAVA"]
